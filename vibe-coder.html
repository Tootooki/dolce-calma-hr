What Is a "Vibe Coder"?

This Role Explained

A "Vibe Coder" is someone who uses AI-powered coding tools — like Antigravity, Claude, Cursor, Replit Agent, or similar — to build automations, scripts, bots, dashboards, and workflows without necessarily being a traditional programmer.

You don't need a CS degree. You need the ability to:

  • Identify repetitive manual tasks across our team
  • Figure out how to automate them using AI tools
  • Build, test, and deploy those automations
  • Continuously find more things to automate
What You'll Be Doing

Your Responsibilities

  • Audit all team workflows and identify automation opportunities
  • Build automations using AI coding tools (Antigravity, Claude, Cursor, Replit)
  • Create scripts, bots, and tools that save the team hours of manual work
  • Integrate APIs: Amazon SP-API, Shopify API, Slack, Google Sheets, Wise, etc.
  • Build Google Sheets automations with Apps Script
  • Set up Zapier/Make.com workflows for cross-platform automation
  • Create Slack bots for team notifications, reporting, and task management
  • Automate reporting: sales dashboards, PPC reports, inventory alerts
  • Build internal tools and dashboards for the team
  • Maintain and debug existing automations when they break
  • Document all automations so the team understands how they work
  • Stay current with the latest AI coding tools and capabilities

Tools You Should Know (or Learn Quickly)

  • Antigravity — AI-powered coding assistant (our primary tool)
  • Claude (Anthropic) — AI assistant for code generation and problem-solving
  • Cursor — AI-powered code editor
  • Replit Agent — Build and deploy apps with AI
  • Google Apps Script — Automate Google Sheets, Docs, Forms
  • Zapier / Make.com — No-code workflow automation
  • Python — For scripts, APIs, data processing (AI writes it for you)
  • GitHub Actions — Automated scheduled tasks and workflows
